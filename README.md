Este projeto é uma aplicação web feita para a disciplina de Arquitetura de Software com o objetivo de mostrar na prática como funciona o padrão MVC usando o Spring Boot 3.x, Java 17 e o Thymeleaf, com o Maven cuidando das dependências.

O código foi dividido no pacote com.exemplo.mvc para deixar a separação da arquitetura bem clara. O Model (Aluno.java) guarda os dados e protege o sistema contra campos vazios usando a validação .isBlank(). O Controller (AlunoController.java) é o cérebro que recebe as requisições do navegador pelas rotas @GetMapping e @PostMapping, salvando os alunos em uma lista na memória e enviando tudo para a tela. A View fica na pasta de templates com os arquivos alunos-form.html e alunos-lista.html, usando o Thymeleaf para montar o formulário e a tabela de alunos de forma dinâmica.

Para rodar o projeto, basta clonar o repositório do GitHub e abrir a pasta raiz no VS Code. O comando para iniciar o servidor é o Run direto na classe principal MvcApplication.java. Com o sistema rodando, é só abrir o navegador no endereço http://localhost:8080/alunos para testar o fluxo completo: você digita os dados no formulário, clica em cadastrar e o Controller processa tudo, te levando direto para a lista atualizada.
